## 尼泊尔 Newar Vajracharya

oṃ āḥ hūṃ

namo ratnatrayāya  
namo guru-vajrasattvāya

buddhaṃ śaraṇaṃ gacchāmi  
dharmaṃ śaraṇaṃ gacchāmi  
saṃghaṃ śaraṇaṃ gacchāmi

dvitīyam api buddhaṃ śaraṇaṃ gacchāmi  
dvitīyam api dharmaṃ śaraṇaṃ gacchāmi  
dvitīyam api saṃghaṃ śaraṇaṃ gacchāmi

tṛtīyam api buddhaṃ śaraṇaṃ gacchāmi  
tṛtīyam api dharmaṃ śaraṇaṃ gacchāmi  
tṛtīyam api saṃghaṃ śaraṇaṃ gacchāmi

bodhicittaṃ utpādayāmi  
sarvasattvānāṃ hitāya sukhāya

oṃ vajrasattva samayam anupālaya  
vajrasattva tvenopatiṣṭha  
dṛḍho me bhava  
sutoṣyo me bhava  
supoṣyo me bhava  
anurakto me bhava  
sarvasiddhiṃ me prayaccha  
sarvakarmasu ca me cittaṃ śreyaḥ kuru hūṃ  
ha ha ha ha hoḥ  
bhagavan sarvatathāgata-vajra mā me muñca  
vajrī bhava mahāsamayasattva āḥ hūṃ phaṭ

ārogya-dīrghāyuḥ-sukha-kṣema-siddhyarthaṃ  
sarvaroga-duḥkha-praśamanārthaṃ  
sarvavighna-praśamanārthaṃ  
sarvapuṇya-jñāna-saṃbhāra-paripūraṇārthaṃ

aparānām āyuḥ-puṇyaṃ ahṛtvā  
aparānāṃ sukha-saṃpadaṃ ahiṃsitvā  
maitrī-karuṇā-muditopekṣā-samanvāgatena cittena  
idaṃ praṇidhānaṃ karomi

oṃ vajrasattva āḥ hūṃ svāhā  
oṃ vajrasattva āḥ hūṃ svāhā  
oṃ vajrasattva āḥ hūṃ svāhā

oṃ namo bhagavate aparimitāyurjñāna-suviniścita-rājāya tathāgatāya arhate samyaksaṃbuddhāya

tadyathā:  
oṃ puṇye puṇye mahāpuṇye  
aparimitapuṇye aparimitapuṇya-jñāna-saṃbhāropacite  
oṃ sarvasaṃskāra-pariśuddhe dharmate gagana-samudgate  
svabhāva-viśuddhe mahānaya-parivāre svāhā

oṃ āḥ hūṃ  
oṃ vajrasattva āḥ hūṃ svāhā

idaṃ puṇyaṃ sarvasattvānāṃ bhavatu  
anuttara-samyak-saṃbodhi-prāptaye

yad atra puṇyaṃ tad bhavatu  
mātā-pitṛ-pūrvāṅgama-sarvasattvānām  
anuttara-jñāna-phala-prāptaye

yat kiñcid aparādhaṃ kāyena vācā manasā vā kṛtam  
tat sarvaṃ vajrasattva-guru-ratnatraya-sākṣikaṃ kṣamayāmi  
sarvaṃ pāpaṃ pratideśayāmi  
sarvaṃ kuśalaṃ anumodayāmi

oṃ vajrasattva āḥ hūṃ svāhā  
oṃ vajrasattva āḥ hūṃ svāhā  
oṃ vajrasattva āḥ hūṃ svāhā

sarve sattvāḥ sukhino bhavantu  
sarve sattvā nirāmayāḥ  
sarve sattvā bhadrāṇi paśyantu  
mā kaścid duḥkhabhāg bhavet

oṃ śāntiḥ śāntiḥ śāntiḥ
