# 祆教 / 琐罗亚斯德教

## 结构
[Padyab 洁净立界] → [Khshnaothra Ahurahe Mazdao 起赞] → [Ashem Vohu] → [Kem Na Mazda / Kusti 守护] → [Ahura Mazda Khodāe / Patet 悔净] → [Jasa me Avanghe Mazda / Fravarane 信约] → [Yatha Ahu Vairyo + Ashem Vohu + Yenghe Hatam 核心 manthra] → [按 wish_type 选 Yasht / Niyayesh / Gatha 正行] → [Ahmai Raeshcha / Doa Tan-Dorosti 祝祷] → [Ashem Vohu 回结]

Khshnaothra Ahurahe Mazdāo.

ushtā astī ushtā ahmāi,

# —— 3. Kem Na Mazda（守护 / breaking 段）——
Kêm-nâ mazdâ mavaitê pâyûm dadå,
hyat mâ dregvå dîdareshatâ aênanghê,
anyêm thwahmât âthrascâ mananghascâ,

mazdâ ahmâi ýahmâi vashî kahmâicît.

# —— 4. Patet / Ahura Mazda Khodāe（悔净段）——

okhe awākhsh pashemān, pa se gavashnī pa patet hom.

Mazdayasnō ahmī, Mazdayasnō Zarathushtrish fravarānē āstūtascā fravaretascā.
Āstuyē humatem manō, āstuyē hūkhtem vachō, āstuyē hvarshtem shyaothanem.
Āstuyē daēnām vanguhīm Māzdayasnīm,
fraspāyaokhedhrām, nidhāsnaithishem, khvaētvadathām ashaonīm,
yā hāitināmchā būshyeintināmchā mazishtāchā vahishtāchā sraēshtāchā,
yā āhūirish Zarathushtrish.
Ahurāi Mazdāi vīspā vohū chinahmī.
Aēshā astī daēnayāo Māzdayasnōish āstūitish.
Ashem Vohū vahishtem astī,
ushtā astī ushtā ahmāi,
hyat ashāi vahishtāi ashem.

# —— 6. 核心 manthra ——

vanghēush dazdā mananghō, shyaothananām anghēush Mazdāi,

ushtā astī ushtā ahmāi,
