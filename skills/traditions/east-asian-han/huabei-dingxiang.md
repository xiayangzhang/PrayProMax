# 华北顶香 / 仙家堂

## 结构
[净口整心] → [报本堂仙家堂] → [报明弟子与愿事] → [胡黄常蟒请仙咒公开段] → [单一目标仙家或本堂公共护持承接] → [陈愿] → [守规止愿] → [回谢] → [送仙归位]

## 通用骨架（占位符版本）
```
【净口整心】
弟子 {{anchors.practitioner_name}}，居于 {{anchors.household_address}}，
今日净口净心，低头敬告本堂仙家堂。

【报本堂仙家堂】
本堂仙家堂诸位老仙家在上，
本堂香烟在前，本堂规矩在心。
弟子 {{anchors.practitioner_name}} 不乱请、不混请、不妄许，
只把 {{anchors.wish_type_name}} 一事报到堂前。

【报明弟子与愿事】
弟子 {{anchors.practitioner_name}} 为 {{anchors.beneficiary}} 陈明：
{{anchors.intention_statement}}
此愿只在 {{anchors.boundary_statement}} 之内，
不伤人，不压人，不夺福，不转灾。

【胡黄常蟒请仙咒公开段】
抓把金丹洒满山，胡黄吃了成大道。
蟐蟒吃了也成仙，留下披毛带甲百草仙。
本堂老仙临香案，本堂老仙听香烟。
老仙家出古洞，离深山，驾清风，入香烟。
今日只请 {{anchors.single_spirit_title}} 当堂受请，
不杂请，不乱请，不越本堂规矩。

【单一目标承接】
愿 {{anchors.single_spirit_title}} 只照 {{anchors.specific_wish}} 一事，
在本堂仙家堂规矩内护持 {{anchors.beneficiary}}。
若此愿合规，愿善缘自来，恶缘自散；
若此愿不合规，愿本堂老仙家拦下，弟子即止。

【守规止愿】
弟子不以此愿求伤害，不以此愿求强取，
不以此愿求强制他人心意，不以此愿求压人财路，
不把堂口名号作恐吓，不把仙家灵验作交易，
不妄许 {{anchors.vow_boundary}} 之外的还愿。

【回谢】
弟子 {{anchors.practitioner_name}} 叩谢本堂诸位老仙家，
叩谢 {{anchors.single_spirit_title}}，
叩谢本堂护持，愿香烟清正，门户清宁。

【送仙归位】
请来有门，送回有路。
{{anchors.single_spirit_title}} 回本位，本堂诸位老仙家各归本位。
香烟落定，堂前清静，门内门外各安其所。
弟子 {{anchors.practitioner_name}} 收愿于心，守规于身。
```
