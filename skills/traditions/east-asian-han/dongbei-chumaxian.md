# 东北出马仙（保家堂口）

## 结构
[净口整心] → [敬本堂掌堂教主] → [报明弟子与所愿] → [请仙咒 / 香堂唱词] → [按愿只请单一目标仙家或本堂公共护持] → [陈愿] → [守规止愿] → [回谢] → [送仙回堂]

## 通用骨架（占位符版本）
```
【净口整心】
弟子 {{anchors.practitioner_name}}，住居 {{anchors.household_address}}，
今日在本堂香案前，净口净心，低头敬告。

【敬本堂掌堂教主】
本堂掌堂教主在上，本堂报马在前，本堂护法兵马在左右。
弟子 {{anchors.practitioner_name}} 不乱请、不妄许、不越堂口，
只把 {{anchors.wish_type_name}} 一事报到堂前。

【请仙咒 / 香堂唱词】
抓把金丹洒满山，胡黄吃了成大道。
老仙家出古洞，离深山，驾清风，入香烟。
文王鼓响通三界，霸王鞭开照堂前。
本堂掌堂教主上香案，本堂护法兵马护四边。
今日只请 {{anchors.single_spirit_title}} 当堂受请，
不杂请，不乱请，不压人，不害人。

【报明所愿】
弟子 {{anchors.practitioner_name}} 为 {{anchors.beneficiary}} 陈明：
{{anchors.intention_statement}}
愿本堂掌堂教主鉴此诚心，
愿 {{anchors.single_spirit_title}} 只在本堂规矩内护持 {{anchors.specific_wish}}。

【守规止愿】
弟子不以此愿求伤人，不以此愿求强取，
不以此愿求夺他人福分，不以此愿求压人财路，
不妄许 {{anchors.vow_boundary}} 之外的还愿，
不把堂口名号作恐吓，不把仙家灵验作交易。

【回谢】
若此愿合堂口规矩，愿善缘自来，恶缘自散；
若此愿不合堂口规矩，愿掌堂教主拦下，弟子即止。
弟子 {{anchors.practitioner_name}} 叩谢本堂掌堂教主，
叩谢 {{anchors.single_spirit_title}}，
叩谢本堂报马与护法兵马。

【送仙回堂】
请来有门，送回有路。
本堂掌堂教主回本位，{{anchors.single_spirit_title}} 回本位，
报马归班，兵马归营，香烟落定，门户清宁。
弟子 {{anchors.practitioner_name}} 收愿于心，守规于身。
```
