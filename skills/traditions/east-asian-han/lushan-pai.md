# 闾山派 / 三奶夫人派

## 结构
[净口净心] → [启告坛前] → [专请临水夫人陈靖姑 / 三奶夫人] → [陈明姓名与所求] → [正行：请陈靖姑咒] → [正行：收惊词] → [安魂归身] → [谢圣回向]

## 通用骨架（占位符版本）
```
净口净心，正念一诚。
弟子{{anchors.petitioner_name}}，为{{anchors.person_name}}，生辰{{anchors.birth_time}}，住居{{anchors.address_or_place}}，今因{{anchors.concern}}，启告坛前。

一心拜请：
临水夫人陈太后，
三奶夫人诸位娘娘，
三十六宫诸位奶娘，
降临香案，鉴察丹诚。

《请陈靖姑咒》
拜请临水陈太后，
三奶夫人到坛前。
左手执剑斩妖邪，
右手执印镇惊惶。
头戴凤冠垂珠翠，
身披霞帔现金光。
护胎护产护男女，
收魂收魄保安康。
弟子一心专拜请，
请到坛前护{{anchors.person_name}}。
若有天惊归天界，
若有地惊归地方；
若有路惊归道路，
若有水火惊归本乡；
若有无名惊与煞，
临水夫人收摄藏。
急急如律令。

再启：
{{anchors.person_name}}三魂七魄，各归本身；胎光、爽灵、幽精，安其本位；尸狗、伏矢、雀阴、吞贼、非毒、除秽、臭肺，各归本宫。

《收惊词》
天惊惊，地惊惊，
弟子{{anchors.petitioner_name}}替{{anchors.person_name}}来收惊。
一收天惊，二收地惊，
三收人惊，四收鬼惊，
五收路惊，六收水惊，
七收火惊，八收刀兵惊，
九收禽兽惊，十收无名惊。
东方惊，东方收；
西方惊，西方收；
南方惊，南方收；
北方惊，北方收；
中央惊，中央收。
惊从门前来，门前收；
惊从路上来，路上收；
惊从水边来，水边收；
惊从火边来，火边收；
惊从梦中来，梦中收；
惊从暗处来，暗处收。
三魂归身，七魄归体；
魂不离身，魄不离体；
魂魄安宁，神气安定。
收收收，收到{{anchors.person_name}}本身中。
急急如律令。

临水夫人陈太后，
三奶夫人诸位娘娘，
愿以此香此词此诚，
护{{anchors.person_name}}身心安定，
惊散魂归，煞退神宁。
弟子{{anchors.petitioner_name}}叩谢圣恩。

所祈{{anchors.concern}}，悉归正道；
若有冤结，愿解不愿结；
若有惊煞，愿散不愿缠；
若有过失，愿忏不愿增。

回向：
愿{{anchors.person_name}}身心安稳，
家宅清宁，
三魂七魄各安其位。
谨请临水夫人、三奶夫人证明。
急急如律令。
```
