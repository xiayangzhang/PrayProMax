# 净土宗

## 结构
[净身/焚香] → [一心顶礼 / 三称本师] → [三皈依] → [忏悔（往昔所造诸恶业…）] → [开经偈 / 诵经（《阿弥陀经》或《无量寿经》节本）] → [赞佛偈（"阿弥陀佛身金色…"）] → [正行：执持名号 "南无阿弥陀佛"（数十声至万声，计数或专注）] → [诵《往生咒》三遍 / 七遍 / 二十一遍] → [发愿（依《莲池大师发愿文》或《慈云忏主净土文》）] → [回向（回向极乐 / 回向法界）] → [三皈依 / 普贤警众偈] → [问讯 / 撤供]

## 通用骨架（占位符版本）
```
（焚香、净手、面西而立或趺坐）

南无本师释迦牟尼佛（三称）
南无西方极乐世界大慈大悲阿弥陀佛（三称）

【三皈依】
自皈依佛，当愿众生，体解大道，发无上心。
自皈依法，当愿众生，深入经藏，智慧如海。
自皈依僧，当愿众生，统理大众，一切无碍。

【忏悔】
往昔所造诸恶业，皆由无始贪嗔痴；
从身语意之所生，一切我今皆忏悔。

【赞佛偈】
阿弥陀佛身金色，相好光明无等伦；
白毫宛转五须弥，绀目澄清四大海；
光中化佛无数亿，化菩萨众亦无边；
四十八愿度众生，九品咸令登彼岸。
南无西方极乐世界，大慈大悲，阿弥陀佛。

【正行：持名念佛】
南无阿弥陀佛  （持诵 {{anchors.recitation_count}} 声，可用念珠计数；专注于佛号，不夹杂、不间断）

【往生咒（拔一切业障根本得生净土陀罗尼）】
（诵 {{anchors.mantra_count}} 遍，梵文罗马化原文）
namo amitābhāya tathāgatāya | tad-yathā oṃ amṛtodbhave | amṛta-siddhaṃ bhave | amṛta-vikrānte | amṛta-vikrānta-gāmini | gagana kīrti-kare svāhā ||

（汉传通用音译）
南无阿弥多婆夜　哆他伽多夜　哆地夜他
阿弥利都婆毗　阿弥利哆　悉耽婆毗
阿弥唎哆　毗迦兰帝　阿弥唎哆　毗迦兰多　伽弥腻
伽伽那　枳多迦利　娑婆诃

【发愿（依慈云忏主净土文，必含"愿生彼国"）】
弟子 {{anchors.practitioner_name}}，一心归命极乐世界阿弥陀佛。
愿以净光照我，慈誓摄我。
我今正念，称如来名，为菩提道，求生净土。
佛昔本誓：若有众生，欲生我国，志心信乐，乃至十念，
若不生者，不取正觉。
以此念佛因缘，得入如来大誓海中，承佛慈力，众罪消灭，
善根增长。若临命终，自知时至，身无病苦，
心不贪恋，意不颠倒，如入禅定。
佛及圣众，手执金台，来迎接我，于一念顷，生极乐国。
花开见佛，即闻佛乘，顿开佛慧，广度众生，满菩提愿。

【个别祈愿（嵌入回向之前，简述即可，不喧宾夺主）】
今为 {{anchors.beneficiary}} {{anchors.intention}}，
惟愿仗此念佛功德，{{anchors.specific_wish}}，
更愿 {{anchors.beneficiary}} 信愿坚固，同生净土。

【回向】
愿生西方淨土中，九品蓮華為父母。
華開見佛悟無生，不退菩薩為伴侶。

愿以此功德，庄严佛净土；
上报四重恩，下济三途苦；
若有见闻者，悉发菩提心；
尽此一报身，同生极乐国。

【普贤警众偈】
是日已过，命亦随减；如少水鱼，斯有何乐？
大众！当勤精进，如救头然；但念无常，慎勿放逸。

南无阿弥陀佛（三称，问讯）
```
