  - 不把红头法师写成黑头道士、僧团、水陆法会或现实中介服务。

# 紅頭法師

## 结构
[净坛 / 设香] → [祝香上告] → [净口 / 净身 / 净天地] → [启师请圣（以本坛祖师为主，不同法脉分段不混请）] → [陈情具疏] → [发愿守禁] → [正行：宝诰 / 神咒 / 科段（收惊、收魂、补运、操营、安营、出煞、安座等）] → [收束法令 / 安魂归位 / 押煞归正] → [谢将退坛] → [回向 / 送圣]

## 通用骨架（占位符版本）
```
【净坛设香】
弟子 {{anchors.practitioner_name}}，为 {{anchors.beneficiary}}，
于 {{anchors.ritual_location}}，谨以清香一炷，肃启本坛。
所陈之事：{{anchors.intention}}。
所涉名籍：{{anchors.beneficiary_name_record}}。
所涉时地：{{anchors.event_time_place}}。

【祝香神咒】
道由心学，心假香传。
香爇玉炉，心存帝前。
真灵下盼，仙旆临轩。
令臣关告，迳达九天。

【净口神咒】
丹朱口神，吐秽除氛。
舌神正伦，通命养神。
罗千齿神，却邪卫真。
喉神虎贲，炁神引津。
心神丹元，令我通真。
思神炼液，道炁长存。
急急如律令。

【净身神咒】
灵宝天尊，安慰身形。
弟子魂魄，五脏玄冥。
青龙白虎，队仗纷纭。
朱雀玄武，侍卫我真。
急急如律令。

【净天地神咒】
天地自然，秽炁分散。
洞中玄虚，晃朗太元。
八方威神，使我自然。
灵宝符命，普告九天。
乾罗怛那，洞罡太玄。
斩妖缚邪，度人万千。
中山神咒，元始玉文。
持诵一遍，却病延年。
按行五岳，八海知闻。
魔王束首，侍卫我轩。
凶秽消散，道炁长存。
急急如律令。

【本坛祖师宝诰（三奶红头通用段）】
志心皈命礼。
闾山学法，道果超伦，妙奥精通。
手持宝剑而灭怪。
玄机洞彻，脚踏白蛇以成神。
忠贞冲日月，义气贯风云。
扶社稷德配乾坤，保苍黎功参造化，孝行克守。
消世上之灾殃，仁爱长存。
施人间之雨露，名登紫府。
果证瑶阙。大悲大愿，至静至仁。
临水宫陈奶夫人。
昭惠崇福护国救世元君。

【金光神咒】
天地玄宗，万炁本根。
广修亿劫，证吾神通。
三界内外，惟道独尊。
体有金光，覆映吾身。
视之不见，听之不闻。
包罗天地，养育群生。
受持万遍，身有光明。
三界侍卫，五帝司迎。
万神朝礼，役使雷霆。
鬼妖丧胆，精怪亡形。
内有霹雳，雷神隐名。
洞慧交彻，五炁腾腾。
金光速现，覆护真人。
急急如律令。

【陈情具疏】
今据 {{anchors.beneficiary}} 所为 {{anchors.wish_type}} 之愿，
并据 {{anchors.specific_condition}}，
不敢越法，不敢妄求，不敢伤害他人。
伏愿本坛祖师鉴明：
若为惊悸失宁，愿 {{anchors.beneficiary}} 三魂安宅，七魄归身；
若为冲煞扰境，愿 {{anchors.place_or_body}} 清宁肃正，凶秽消散；
若为补运植福，愿 {{anchors.beneficiary}} 正路开明，福缘归善；
若为安座安营，愿 {{anchors.ritual_object_or_place}} 定位安宁，内外和合；
若为特定事件，愿 {{anchors.event_name}} 平正无碍，善缘具足。

【发愿守禁】
弟子 {{anchors.practitioner_name}} 今于坛前明愿：
不以此法伤人，不以此法夺魂，不以此法制人情志，
不以此法求横财，不以此法替代医药救护，
不混请别坛祖师，不泄本坛符诀。

【正行科段占位】
本段由 {{anchors.officiant_id}} 依 {{anchors.ritual_section_name}} 成文；
凡符令、手诀、步罡、兵马、收魂、出煞、安营、安座等密传内容，
只保留为坛内法令，不在公开祷词中摹写。

【收束】
愿 {{anchors.beneficiary}} 魂魄安身，心神归正；
愿 {{anchors.ritual_location}} 内外清宁，冲煞退散；
愿 {{anchors.intention}} 归于正善，不犯禁忌。

【谢将退坛】
香云已达，疏意已陈。
有请有送，有启有谢。
本坛祖师慈悲鉴纳，诸司将吏各归本位。
急急如律令。

【回向】
愿以此清净功德，回向 {{anchors.beneficiary}}，
回向 {{anchors.related_people}},
回向 {{anchors.place_or_household}} 内外安宁，
诸恶不作，众善奉行，怨结消释，福善增长。
```
