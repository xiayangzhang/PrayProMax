# 瑶族度戒 / 挂灯

## 结构
[启坛净场] → [请师请圣] → [报坛陈愿] → [挂灯授戒] → [领戒应诺] → [谢师送圣] → [回向祖先与家门]

## 通用骨架
```
伏以，{{anchors.place}}坛前，今有瑶家弟子{{anchors.initiand_name}}，年庚{{anchors.birth_info}}，家门{{anchors.family_line}}，随师登坛，度戒挂灯。

《度戒唱词·启坛》
开坛开路，净坛净场；
香烟上达，灯火分明。
今日良辰，立坛度戒；
弟子{{anchors.initiand_name}}，跪在坛前。

《度戒唱词·请师》
请祖师临坛。
请师请圣到坛前，
度戒挂灯照本元。
弟子{{anchors.initiand_name}}来受戒，
师法分明记根源。

请盘王祖先临坛。
盘王在上，家先在前；
照见子孙，认祖归源。
灯挂坛中，名入师门；
戒牒分明，护佑家门。

请本坛师法临坛。
师公引路，法灯开明；
一灯照身，一灯照名。
戒门今日开，弟子今日承；
{{anchors.initiand_name}}稽首，愿守师门。

《度戒唱词·报坛陈愿》
弟子{{anchors.initiand_name}}，本命{{anchors.life_anchor}}，家住{{anchors.home_place}}。
今为{{anchors.rite_intent}}，登坛度戒，挂灯受名。
不忘祖先，不背师法；
不乱戒条，不坏坛门。

《度戒唱词·挂灯授戒》
灯起坛中，光照本命；
灯挂师门，名归法籍。
上灯照天门，
中灯照人身，
下灯照家宅。
{{anchors.initiand_name}}受灯，{{anchors.initiand_name}}受戒，{{anchors.initiand_name}}受师门法名{{anchors.dharma_name}}。

《度戒唱词·领戒应诺》
师问：守祖先否？
弟子答：守。
师问：敬师法否？
弟子答：敬。
师问：持戒条否？
弟子答：持。
师问：不乱坛门否？
弟子答：不乱。
师问：不害家人、不害乡邻否？
弟子答：不害。

《度戒唱词·谢师送圣》
香烟已达，灯位已明；
戒名已立，师法已承。
谢祖师，谢盘王，谢家先，谢本坛师法。
来时有路，去时有程；
护持{{anchors.initiand_name}}，护持{{anchors.family_line}}家门。

《度戒唱词·回向》
所修度戒挂灯功德，
回向祖先家门，
回向{{anchors.initiand_name}}身命清明，
回向{{anchors.family_line}}根源不断。
灯明不灭，师法长存。
```
