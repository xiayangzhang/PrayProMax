# 苗族椎牛仪 / 还傩愿

## 结构
[净坛与报寨] → [启坛请傩] → [告祖叙源] → [陈愿主名分] → [椎牛献牲 / 象征献牲] → [还傩愿正唱] → [祛秽送煞] → [谢祖谢傩] → [收坛回向]

## 通用骨架（占位符版本）
```
【净坛报寨】
今夜开坛，今朝明愿。
{{anchors.village}}寨中，{{anchors.household}}家门，
愿主{{anchors.petitioner_name}}，旧愿不敢忘，新愿不敢乱。
坛门打开，秽路关起；正路进坛，邪路退避。

【启坛请傩】
傩公在上，傩母在上。
请傩公下坛，请傩母临坛。
旧年所许，今日来还；
口中所许，手中来交；
心中所愿，坛前来明。
傩公莫怪，傩母莫嗔，
愿主{{anchors.petitioner_name}}叩头还愿。

【告祖叙源】
祖公在上，祖婆在上。
{{anchors.clan_name}}门中祖先，
从老根来，从老路来，
从高山来，从旧寨来，
今朝听鼓，今朝听歌，
听巴代开口，听儿孙报明。
儿孙{{anchors.petitioner_name}}不忘祖根，
家门{{anchors.household}}不乱祖路。

【陈愿主名分】
巴代唱：
哪家许愿？——{{anchors.household}}许愿。
哪人还愿？——{{anchors.petitioner_name}}还愿。
为何许愿？——为{{anchors.wish_reason}}许愿。
何日来还？——今日{{anchors.ritual_date}}来还。
愿有头，愿有尾；
话有根，话有证；
请傩听明，请祖听明。

【椎牛祭祀唱词 / 献牲段】
牛头向坛，牛角向天。
牛蹄踏地，牛尾扫尘。
不是空口来还愿，
不是虚言来开坛。
牲到坛前，愿到坛前；
血不乱流，名不乱报；
献给祖先，祖先受明；
献给傩坛，傩坛受明。
若以象征牲礼代行，
则以{{anchors.symbolic_offering}}当牲名，
以净水当明路，
以香火当报信，
以歌声当还愿。

【还傩愿正唱】
还愿还愿，旧愿今日还。
许在{{anchors.vow_time}}，还在{{anchors.ritual_date}}。
许时求{{anchors.wish_reason}}，
还时谢傩恩。
傩公坐中堂，傩母坐上位；
愿主叩头，家门低声；
一还口愿，二还心愿，三还家门愿。
愿不欠坛，坛不压人；
话不欠神，神不追愿。
旧结今日解，旧愿今日清。

【祛秽送煞】
门前煞，送出门；
路上煞，送下路；
病根煞，断其根；
口舌煞，闭其门。
不留在{{anchors.household}}，
不缠{{anchors.petitioner_name}}，
不入祖堂，不进火塘。
去者去远，散者散清。

【谢祖谢傩】
谢傩公，谢傩母；
谢祖公，谢祖婆。
听了今夜歌，受了今日愿。
愿主{{anchors.petitioner_name}}还愿已明，
家门{{anchors.household}}名分已清。
祖路归祖路，傩坛归傩坛；
鼓声收鼓声，歌声收歌声。

【收坛回向】
坛门合，愿路清。
明处归明处，暗处归暗处。
所还之愿，归于{{anchors.wish_dedication}}；
所谢之恩，归于{{anchors.clan_name}}门中。
巴代收词，愿主叩首。
今日还傩愿，至此收坛。
```
