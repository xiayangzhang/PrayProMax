# 土家族梯玛 / 还愿

## 结构
[净口开坛] → [报坛告名] → [请家堂祖先] → [请土王或本坛主神] → [陈愿名与愿主] → [还愿正歌] → [献酒饭牲礼名目] → [谢祖谢神] → [解愿清口] → [送神闭坛]

## 通用骨架（占位符版本）
```
梯玛开口，净口净心；
掌坛梯玛，立在坛前。
今日开坛，今日告愿；
愿主名姓，{{anchors.petitioner_name}}；
家门地址，{{anchors.place_name}}；
所许愿名，{{anchors.vow_name}}。

列祖列宗，回堂受香；
家先祖公，家先祖婆；
上代祖宗，下代子孙；
香烟一炷，酒饭一堂。

土王在上，土王坐坛；
本境土王，本寨土王；
听我梯玛，听我神歌；
今日还愿，愿有根名。

向王天子，请临神位；
香烟请到，酒饭请到；
愿主{{anchors.petitioner_name}}，
曾因{{anchors.vow_reason}}，
口许{{anchors.promised_offering}}，
心许{{anchors.promised_act}}；
今日备办{{anchors.actual_offering}}，
今日唱明{{anchors.actual_vow_return}}。

许愿有根，还愿有名；
许愿不空，还愿不虚；
酒到坛前，饭到坛前；
粑粑到坛前，清茶到坛前；
{{anchors.offering_items}}，一一报明。

梯玛神歌，唱上三声；
一声报祖，二声报神；
三声报愿，愿清愿明。
愿主所求，{{anchors.wish_statement}}；
愿主所谢，{{anchors.thanks_statement}}；
旧愿今日清，新愿不乱许。

祖先受香，土王受酒；
受了香烟，受了酒饭；
护家门，护火塘；
护老少，护田土；
不添口舌，不添惊扰。

若有错名，今日改正；
若有漏愿，今日补明；
若有冲犯，今日解开；
若有拖欠，今日还清。
愿从坛上解，话从口中清；
清清白白，明明朗朗。

谢祖先，祖先归位；
谢土王，土王归位；
谢诸坛神，诸坛神归位。
请神来，送神去；
来有香烟，去有路程。
坛门收起，愿事收圆。
梯玛收声，愿主叩谢。
```
