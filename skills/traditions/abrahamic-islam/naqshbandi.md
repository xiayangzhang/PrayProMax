mitigation: |
  须先 wudu'（洗净）并面 qibla；以 Bismillah 起、以 salawat + Al-Fatiha 回向先知与 silsila；
  不得绕开 shaykh 行 tawajjuh 或自创 wird；念毕将功德归于 ruh of Prophet ﷺ、

taboos:

  - 不得跳过 silsila 直接呼请高阶 awliya
  - 不得删改或翻译 wird 的原文阿拉伯/波斯主体
  - 月经 / 产后 / 不洁状态下不触 Qur'an 原本、不进行正式 dhikr 坐法
  - 不得收取金钱出售 ijaza（修法授权）
  - 不与异性独处行 tawajjuh
  - 擅自加入诅咒、伤害性咒法、跨宗教神名或非 Naqshbandi occult 元素
  - 公开宣扬秘密教义
  - 无指导下进行高级 khalwa

# 苏菲 Naqshbandi

## 结构
[Niyya 立意（静默）] → [Ta'awwudh 求护] → [Bismillah 起] → [Al-Fatiha 一遍] → [Salawat 三遍] →
[Rabita / Tawajjuh 心向 shaykh 与 silsila（仅在已 bay'a 时）] → [正行 Dhikr khafi —— Kalima
"La ilaha illa Allah" 或 Ism al-Dhat "Allah" 默念定数] → [Wird al-Hajat 具体祷求（按 wish_type
选 Asma' al-Husna / Surah / hizb）] → [Du'a 用阿拉伯语自由祈愿] → [Salawat 三遍 + Al-Fatiha 一遍]
→ [Isal al-thawab 回向（先知 ﷺ → Khwajagan 历代 → silsila → 亡者与众生 → 本祷对象）]
