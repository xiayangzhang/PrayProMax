# Ngillatun / Machi

## 结构
问安与自报来处 → 分段呼请 Ngünechen、Ñuke Mapu、pu ngen 与 kuifikecheyem → 陈明愿心 → llellipun / tayül 式正行 → 互惠承诺 → 平衡与回返
