# 长屋宗教（Longhouse / Handsome Lake）

## 结构
[问安与收心] → [说明同心同意] → [Ohén:ton Karihwatéhkwen / Ganö:nyök：向 People 致谢] → [向 Mother Earth 致谢] → [向 Waters 致谢] → [向 Fish / Plants / Food Plants / Medicines 致谢] → [向 Animals / Trees / Birds 致谢] → [向 Four Winds / Thunderers 致谢] → [向 Sun / Grandmother Moon / Stars 致谢] → [向 Enlightened Teachers / Creator 致谢] → [以 “our minds are one” 收束] → [如属 Gaihwi:io 场合，由授权 speaker / faithkeeper 接续伦理训诫或仪式正文]
