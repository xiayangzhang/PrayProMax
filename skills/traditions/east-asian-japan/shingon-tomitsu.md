# 真言宗（東密）

## 结构
懺悔 → 三帰依 → 発菩提心 → 本尊勧請 → 正行真言念誦 → 所願言上 → 回向

## 通用骨架
```
懺悔：
我昔所造諸悪業　皆由無始貪瞋癡
従身語意之所生　一切我今皆懺悔

三帰依：
南無帰依仏　南無帰依法　南無帰依僧

発菩提心：
衆生無辺誓願度　煩悩無尽誓願断
法門無量誓願学　仏道無上誓願成

本尊勧請：
南無大日如来
namas samanta-buddhānāṃ a vi ra hūṃ khaṃ

正行真言：
oṃ vajrasattva vaṃ
namas samanta-buddhānāṃ a vi ra hūṃ khaṃ
oṃ amogha vairocana mahāmudrā maṇipadma jvāla pravarttaya hūṃ
gate gate pāragate pārasaṃgate bodhi svāhā

所願言上：
弟子 {{anchors.practitioner_name}}、三宝・大日如来の前に、{{anchors.wish_intent}} のため、身口意を調え、正法に違わず、{{anchors.target_name}} に関わる障碍の消滅と善縁の成就を願い奉る。

回向：
願以此功徳　普及於一切
我等与衆生　皆共成仏道

南無大師遍照金剛
```
