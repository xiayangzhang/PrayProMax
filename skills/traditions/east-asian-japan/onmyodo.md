# 陰陽道

## 结构
[斎戒・祓] → [神名奉称] → [本命星・当年星の申告] → [発願] → [正行：星祭祭文・反閇咒・急急如律令] → [鎮謝] → [回向・納め]

## 通用骨架（占位符版本）
```
掛けまくも畏き祓戸大神等、
{{anchors.practitioner_name}} が身に負ひ持つ罪穢れを、
祓へ給ひ清め給へと白す。

高天原に神留まり坐す。
皇親神漏岐神漏美の命以ちて、
八百万神等を神集へに集へ給ひ、
神議りに議り給ひて、
我が皇御孫命は豊葦原瑞穂国を、
安国と平けく知ろし食せと事依さし奉りき。

天地陰陽の道を司り給ふ御神に白す。

泰山府君大神、恐み恐みも白す。
{{anchors.practitioner_name}}、{{anchors.birth_year}} 生まれ、
本命星 {{anchors.honmyo_star}}、当年星 {{anchors.tonen_star}}、
住所 {{anchors.location}} に在りて、
{{anchors.wish_content}} の願意を捧げ奉る。

北辰妙見尊、恐み恐みも白す。
星辰順度、陰陽和合、災厄消除を乞ひ奉る。

十二天将、恐み恐みも白す。
青龍、朱雀、勾陳、騰蛇、白虎、太常、玄武、太陰、天后、六合、太裳、天空、
方位の障り、日時の障り、物忌の障りを鎮め給へ。

星祭祭文の段。
謹請、北辰妙見尊。
謹請、本命星君。
謹請、当年星君。
謹請、九曜星君。
謹請、二十八宿星君。
願以此功徳、星辰清明、災厄消除、福寿増長、家門安穏。
急急如律令。

反閇咒の段。
天罡地煞、歩罡踏斗。
左青龍、右白虎、前朱雀、後玄武。
臨兵闘者皆陣列在前。
急急如律令。

破障・解除の段。
若有悪咒、若有邪気、若有方災、若有星厄、
返本還源、解除消滅。
急急如律令。

鎮謝の段。
奉謝、泰山府君大神。
奉謝、北辰妙見尊。
奉謝、十二天将。
奉謝、本命星君、当年星君、九曜星君、二十八宿星君。

回向の段。
願以此功徳、
普及於 {{anchors.dedication_target}}、
陰陽和順、星宿清明、災厄消除、所願成就。
恐み恐みも白す。
```
