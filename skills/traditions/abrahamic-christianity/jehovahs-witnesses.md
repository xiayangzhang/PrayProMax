# 耶和华见证人

## 结构
[称呼耶和华为天父] → [称颂并尊崇耶和华的名] → [承认王国与旨意优先] → [为会众或个人的属灵需要、日用需要、宽恕、智慧、忍耐与保护作祈求] → [感谢耶和华] → [奉耶稣基督的名结束] → [阿们]
