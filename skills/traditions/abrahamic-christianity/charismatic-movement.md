# 灵恩派（Charismatic Movement）

## 结构
[敬拜与归向天父] → [主祷文或圣经宣读] → [认罪与饶恕] → [弃绝罪、偶像、惧怕与不义约束] → [奉耶稣基督之名宣告福音与十字架得胜] → [抵挡邪恶、命令不洁之灵离开] → [求圣灵充满、医治与更新] → [为具体事项代求] → [感恩、交托与阿们]
