# 托雷斯海峡 Ailan Kastom

## 结构
[承认岛、海与家族来处] → [向 bipo pipol / bigwan 致敬] → [说明本祈愿不取用受限歌咒] → [正行：公开 Yumplatok 祈愿唱诵] → [立愿守 kastom 与 wanwan] → [收束：eso / amen]

## 通用骨架（占位符版本）
```
Mipla stanap long ples blo solwata, win ane ailan.
Mipla tingbaut bipo pipol, ol bigwan, ane kastom we i bin kipem mipla.
Mipla no tekem song blo nada pamle.
Mipla no kolem nem blo man, ailan, reef, zogo o stori wea mipla no gad lo.

Papa God, yu lukaut mipla.
Yu lukaut {{anchors.person}}.
Yu lukaut pamle blo {{anchors.family}}.
Yu lukaut {{anchors.place}} ane ol pipol uda stap de.
Meke mipla strong.
Meke mipla gudwei.
Meke mipla wanwan.
Tekwei nugud tingting, nugud tok, nugud pasin.
Kipem mipla long stret rod blo kastom.

Bipo pipol, mipla tingbaut yu.
Bigwan, mipla lisen long gud tok.
Mipla kipem respek.
Mipla kipem pamle.
Mipla kipem ailan ane solwata.

Eso.
Amen.
```
