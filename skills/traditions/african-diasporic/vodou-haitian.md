# —— 安全与规范 ——
backlash_risk: high
mitigation: "必先开 Legba 之门方可呼请其余 lwa；严格区分 Rada（凉）与 Petwo（热）两族，不可同段呼请；任何向 lwa 许诺的供奉（manje-lwa、kleren、动物、彩巾）必须按期兑现，违诺反噬称 \"dèt lwa\"；涉及 wanga / pwen 等带攻击性之工，必须由 houngan / manbo 操作并自行承担 reprezye；限用公开 Priye Ginen、公开 lwa 颂名与非伤害性祷词，不写秘密 langaj、initiation passwords、动物祭献步骤或攻击性 wanga。"

taboos:
  - "未先呼 Papa Legba（开门）即直接呼其他 lwa。"
  - "同一段祷词中混合 Rada 与 Petwo 族 lwa，或在同一正行段混合多个 lwa 的直接呼请。"
  - "向 Gede（亡灵族）祈求时不戴黑紫、不奉 kleren 浸 21 种辣椒。"
  - "周三 / 周六（Petwo 日）行 Rada 之 sèvis，反之亦然。"
  - "不按期 / 不按种类兑现对 lwa 的供奉承诺（招 dèt lwa）。"
  - "以不当供品喂 lwa（如某些 Rada lwa 忌盐）。"
  - "由非 kanzo 之人独立使用 Langaj 中保留给启灵者的 mò sekrè（秘语）。"
  - "将 lwa 等同于天主教圣徒本体（仅可作 koresponadans 对应，不可替代）。"
  - "在月经期 / 丧期内进入 hounfò 主祭坛或操作 asson。"
  - "拍摄 / 录音 sèvis 中 possession 状态而未先获 lwa 与主持同意。"
  - "跨 sosyete 直接挪用他门派 chante / langaj 而未经 transfert。"
  - "把 Haitian Vodou 与 Lucumí / Santería、Palo、New Orleans Voodoo 或西方咒术任意混合，未经家门同意挪用跨传承元素。"
  - "伪造 veve、secret langaj、asson lineage 密语、kanzo 权限或亡者召回细节。"
  - "写诅咒、强迫爱情、报复、伤害、尸骨 / 墓地操作或现实动物祭献指令。"
  - "把 Bondye 当作可被交易操控的对象；lwa 为服务与互惠关系，不作交易勒索。"
  - "不尊敬祖先或未经引导穿过 / 干预他人 practitioner 之 wanga / pwen 工作。"

# 海地伏都 Vodou Haïtien

## 结构

完整 sèvis 结构（顺序不可调换）：

[Priye Katolik（天主教起首段：Pater Noster → Ave Maria → Credo → Confiteor）] →
[Priye Dyò / Priye Ginen（呼请 Ginen 祖灵之地与历代 mò、misté、marasa）] →
[呼 Papa Legba 开 barye（门）] →
[按族系次第 salut nasyon（Rada → Nago → Kongo → Ibo → 最后 Petwo / Gede，热冷分段）] →
[呼请本次 sèvis 主 lwa（chante + Langaj reskonsab）] →
[发愿 / 陈请（priyè patikilye）] →
[供奉 manje-lwa / kleren / 蜡烛 / 彩巾] →
[（如有）possession / reskonsab 之口传] →
[回向 / 谢礼（remèsiman）] →
[关 barye（Ayibobo 三呼，送 Legba 闭门）]

个人 priyè 简化结构（俗信可自念部分）：

[十字圣号] → [Pater + Ave + Credo] → [呼 Papa Legba 开门] → [呼单一对应 lwa（Rada **或** Petwo，不混）] → [陈请] → [承诺供奉] → [Ayibobo 三声闭]
